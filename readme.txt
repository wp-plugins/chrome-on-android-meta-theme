Adds the meta-theme tag to every page to change the toolbar color in Chrome on Android

== Description ==

Starting in version 39 of Chrome for Android on Lollipop, you can use the theme-color meta tag to set the toolbar color â€” this means no more Seattle gray toolbars!


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `chrome-meta-theme` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Settings > Chrome Meta Theme
1. Add the main hex color of your theme i.e. #dd4b39
